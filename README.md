
# SIGE

Trabalho desenvolvido para a matéria de Tópicos Especiais II.




## Instalação


Clone o repositório do Github
```bash
  git clone https://github.com/fabriciojuniors/te2-sige
```

Acesse a pasta do projeto
```
  cd te2-sige
```

Instale as dependências do projeto 
```
  npm install 
```

Execute a aplicação
```
  ionic serve   
```
